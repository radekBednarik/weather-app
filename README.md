# Weather app

React Next.js web app to get weather forecast for your location.

## Development

### Getting Started

Install

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
